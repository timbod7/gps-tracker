A prototype GPS tracker, ultimately intended for on-water use

# Wiring

## GPS

* `green (rx)` -> `pa9 (tx)`
* `blue (tx)` -> `pa10 (rx)`

## LS027B7DH01 Display

*   pa4 - CS
*   pa5 - SCLK
*   pa7 - MOSI
